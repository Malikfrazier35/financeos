Infrastructure

Built on best-in-class, SOC 2 Type II certified providers — never operating our own data centers.

Application hosting
Vercel (SOC 2 Type II, ISO 27001). US-East infrastructure with global edge distribution. Zero-downtime deploys, automatic SSL, DDoS protection.
Database
Supabase (SOC 2 Type II, HIPAA-ready). Postgres 15 with row-level security enforced at the database layer. Automated daily backups with 30-day retention.
Payments
Stripe (PCI-DSS Level 1, SOC 2 Type II). We never store credit card numbers — all payment data is tokenized and held by Stripe directly.
AI processing
Anthropic Claude API (SOC 2 Type II). Your data is never used to train models. Responses are processed in-memory and not persisted by the AI provider.
