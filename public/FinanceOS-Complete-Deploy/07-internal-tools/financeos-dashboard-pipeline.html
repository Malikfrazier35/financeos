All 13 Dashboard Views

Each sidebar nav item maps to a view ID and a feature key. The plan gate checks the user's Stripe-verified tier against the feature key. If locked, an upgrade prompt renders instead of the view content.
Dashboard Overview
6 KPI cards (ARR, NDR, Gross Margin, Rule of 40, Burn Multiple, Headcount), revenue chart, department spend donut, activity feed, AI insights bar.
● Live — Full interactivity
AI Copilot
Claude-powered financial assistant with visible chain-of-thought reasoning. Chat interface, suggested prompts, connects to live financial data across 7 systems.
● Built — Claude API integrated
P&L Statement
Full income statement: Revenue breakdown (Subscription, Services, AI Module), COGS, Gross Profit, OpEx by department, EBITDA. Drill-down drawers on every line item.
● Built — Clickable drill-downs
Forecast Engine
ML-powered 12-month revenue forecast with confidence intervals, sensitivity sliders (churn, expansion, pipeline), actual vs. predicted tracking, MAPE accuracy scoring.
● Built — ML model + sliders
Scenario Modeling
3-scenario comparison engine (Base, Optimistic, Conservative) with adjustable drivers: growth rate, churn, headcount, burn. Exportable board-ready outputs.
● Built — 3 scenarios + export
Multi-Entity Consolidation
Acme US, Acme EU (EUR), Acme APAC (SGD). Intercompany eliminations, FX translation, entity-level close status, drill-down by entity with currency conversion.
● Built — 3 entities + FX
Close Task Manager
Month-end close workflow: checklist items with assignees, due dates, status tracking (done/in-progress/pending). Revenue recognition, reconciliations, accruals.
● Built — Full task workflow
Team Management
Team member list, roles, activity chat (Slack-like), task assignments, real-time presence indicators. Role-based access for CEO/CFO/Controller views.
● Built — Chat + presence
Integrations Hub
Data connector marketplace: QuickBooks (live OAuth), Stripe, Salesforce, HubSpot, Netsuite, Workday, Slack. Sync status, last-sync timestamps, health monitoring.
● Built — QuickBooks OAuth live
Investor Metrics
Series A readiness scorecard (8/8 benchmarks), SaaS metrics grid, cohort retention heatmap, fundraising readiness bar. CSV/PDF export for board packs.
● Built — Board-ready exports
Command Center
Executive hub with role-based tabs (CEO, CFO, Controller). AI insights carousel, deal pipeline, activity feed. CEO: pipeline coverage + win rate. CFO: cash flow + margins. Controller: close checklist + SOX.
● Built — 3 role dashboards
Admin / Settings
Workspace configuration, connected systems, API key management, notification preferences, data export controls, team permissions.
● Built — Full settings panel

Remaining Pipeline — What Needs to be Built
Priority 1 — High Impact
SOX Audit Trails View
Dedicated view for SOX compliance: audit log table with timestamps, user actions, control sign-offs, automated evidence collection. Currently only partially referenced in the Controller tab of Command Center.
Feature key: sox_audit · Business tier
Priority 2 — Revenue Enabler
SSO / SAML Integration
Enterprise SSO via SAML 2.0 or OIDC. Requires Supabase Auth extension or custom identity provider hook. Admin UI for configuring IdP settings, domain verification, and user provisioning.
Feature key: sso · Business tier
Priority 3 — Differentiation
Custom ML Models
UI for uploading training data, selecting model type (forecast, anomaly detection, classification), training status dashboard, A/B model comparison, and deployment to live forecasting pipeline.
Feature key: custom_ml · Business tier
Priority 4 — Growth Tier
Live Slack OAuth
The Slack integration card exists in the Integrations view but has no live OAuth flow. Needs: Slack App creation, OAuth callback route, channel selector, and notification rules (budget alerts, close reminders).
Feature key: slack_integration · Growth tier
Priority 5 — Polish
Spring Themes → Vercel Deploy
3 Spring 2026 templates (Sage & Stone, Botanical, Executive Blossom) + theme chooser page are built and in the public/ folder. Need to push to GitHub to trigger Vercel auto-deploy to finance-os.app.
Routes: /themes, /sage-stone.html, /botanical.html, /executive-blossom.html
Priority 6 — Data
Live Data Connections
Most views use hardcoded demo data. Connecting to live Supabase tables (already provisioned) would make P&L, Forecast, and Consolidation views pull real financial data. QuickBooks is the only live integration so far.
Supabase tables: transactions, forecasts, entities, audit_log
