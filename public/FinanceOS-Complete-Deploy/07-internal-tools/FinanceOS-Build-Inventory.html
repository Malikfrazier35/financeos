Section 2

Key improvements in this build cycle

Everything that changed from the original dark-theme pages to the current light-theme deployment candidates.

01

Dark to light theme conversion

All new pages use a clean white and light gray design system. Matches the industry standard for enterprise B2B financial software marketing sites.

02

Frosted-glass navigation

Frosted glass navigation bar that stays visible at the top of the page as visitors scroll. Transparent with a soft blur effect. Consistent across every page.

03

5-tier shadow hierarchy

Five levels of visual depth (from subtle to elevated) replacing a single flat shadow. Creates proper layering for cards, overlays, and featured sections throughout the site.

04

Typography system overhaul

Two professional font families: DM Sans for all marketing copy, JetBrains Mono for financial data and metrics. Headlines use tight letter-spacing for impact. Category labels use wide spacing for clarity. 5 distinct text color tiers from headings to muted captions.

05

Capability grid layout

Asymmetric 3-column grid with double-width emphasis on key capabilities (AI Copilot, Integrations). Replaces flat card lists. Each card features KPI stats and mini-charts.

06

Dashboard product visual

Full application mockup in hero: sidebar with 8 nav items, 4 KPI metric cards, 12-bar chart. Shows the product instead of describing it.

07

AI Copilot demonstration

Interactive mockup showing a real AI-driven variance analysis conversation (OPEX budget overrun with sourced line items). Shifts from "we have AI" to "here's what our AI does for your finance team."

08

Real connector brand logos

Integrations V3 replaced generic letter abbreviations with actual brand-colored logo marks for all 27 connectors (NetSuite, SAP, Salesforce, Stripe, etc.).

09

Social proof and metrics strips

8-company trust bar (Figma, Notion, Vercel, Linear, Ramp, Mercury, Retool, Webflow) plus 5 operating metrics ($48.6M ARR, 118% NDR, 2,400+ Teams, 0.8x Burn Multiple, 84.7% Gross Margin). Standard enterprise SaaS credibility pattern.

10

Scroll-triggered animations

Page sections fade in as visitors scroll, creating a guided experience instead of a static wall of content. Consistent across all light-theme pages.

11

Dual conversion paths

Every major section offers 2 clear next steps: primary (Start Free Trial) + secondary (Book Demo / Talk to Sales). Captures both self-serve buyers and enterprise prospects who prefer guided sales.

12

Mobile and tablet responsive layouts

4 responsive layouts across all pages: desktop (full grid), tablet (two-column), mobile (single-column), small phone (stacked metrics). Consistent behavior everywhere.

13

Pain/solution visual pattern

Side-by-side comparison: red-accented "pain" cards (showing the problem) next to green-accented "solution" cards (showing how Castford fixes it). Used across Pain-Points and Solutions pages.

14

Category filtering

Interactive filter buttons for Integrations (7 categories: ERP, CRM, HRIS, Banking, Data, BI), Resources (5 types), and Solutions (6 segments). Resources page also includes live keyword search.

Pending work before full deploy

Remaining tasks to get from prototype HTML files to production-ready pages on castford.com.

All 5 pages (Pricing V3, Competitors, xP&A, Enterprise Pricing, Ad Campaign) converted to light theme with frosted-glass nav, light hero, and consistent footer.

Completed

Cross-link navigation

All 15 pages now have consistent nav with real cross-links: Solutions, Integrations, Trust, Pricing, Compare, Resources | Sign In, Subscribe.

Completed

Match live site nav structure

Nav updated to: Solutions, Integrations, Trust, Pricing, Compare, Resources. Shared template build script (build.js) manages nav/footer from a single source.

Medium Priority

Brand logos on remaining pages

Only Integrations V3 has full brand logo marks. Landing V3 social proof strip and other pages still use text-only company names.

Medium Priority

Mobile navigation menu

Navigation links currently hide on mobile devices with no toggle menu. Need a collapsible mobile menu for phone and tablet visitors.

Medium Priority

Lead capture and form submission

Call-to-action buttons and email fields are visual-only. Need to connect to a lead capture service or CRM so trial signups and demo requests are actually received by the sales team.

Lower Priority

Dark mode toggle

Current live site has a dark mode toggle. V3 pages are light-only. Could add a theme switcher to match existing site functionality.

Lower Priority

SEO meta tags

Pages need social sharing previews, search engine descriptions, and proper page URLs for production deployment.

Lower Priority

Analytics integration

Add analytics to measure visitor behavior: page scroll depth, call-to-action clicks, trial signups, and demo request conversions.
