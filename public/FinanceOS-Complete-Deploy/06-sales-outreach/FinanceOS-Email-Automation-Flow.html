P0 — No Conversion Tracking Tag on Site
Google Ads created a "Submit lead forms" conversion action, but the tracking tag was never added to castford.com. Google can't report conversions, can't optimize bidding, and your $30/day is spending blind. Performance Max relies on conversion signals to find the right audience — without them, it's guessing.
Fix: Install the Google Ads gtag.js snippet + conversion event on your form's thank-you page or form submission handler
P0 — Duplicate Email Sends
Krista Lumpkin received 4-5 copies of both confirmation and welcome emails. This is likely caused by: (a) the form submit handler firing multiple times on click (no debounce), (b) the email trigger running on every page load if there's a query param, or (c) a race condition in whatever automation tool sends the emails.
Fix: Add form submission debounce, disable the button after first click, and implement idempotency keys in your email sender
P1 — No Email Sequence After Signup
After the initial confirmation + welcome, there's no nurture sequence. Leads go cold. You're paying $30/day to acquire them and then abandoning them. The 88 clicks this week generated leads that aren't being nurtured toward demo or purchase.
Fix: Build a 7-email drip sequence over 14 days with increasing urgency and social proof
P1 — Final URL Expansion Turned Off
Google is recommending you turn on Final URL expansion. This lets Performance Max dynamically route people to the most relevant landing page based on their search. With it off, everyone lands on the same page regardless of intent.
Fix: Enable Final URL expansion in campaign settings (it's an "AI Essential")
P2 — No Customer Match List Uploaded
Google recommends uploading a Customer Match list to help Performance Max find similar audiences. If you have any existing email list (beta signups, waitlist, etc.), uploading it would dramatically improve targeting quality.
Fix: Export your signup list and upload as Customer Match in Google Ads audience manager
Google Ads Conversion Tracking — Implementation Guide

Step 1: Add the Global Site Tag (gtag.js)

Add this to your app's <head> section in page.jsx or _document.jsx. Replace AW-XXXXXXXXX with your Google Ads conversion ID from your Google Ads account.

<!-- Google Ads Global Site Tag -->
<script async src="https://www.googletagmanager.com/gtag/js?id=AW-XXXXXXXXX"></script>
<script>
  window.dataLayer = window.dataLayer || [];
  function gtag(){dataLayer.push(arguments);}
  gtag('js', new Date());
  gtag('config', 'AW-XXXXXXXXX');
</script>

Step 2: Fire the Conversion Event on Form Submit

In your form submit handler (the demo request form or waitlist form), fire the conversion event AFTER successful submission. Replace AW-XXXXXXXXX/YYYYYYY with your conversion label.

// In your form submit handler, after successful API response:

const handleSubmit = async (e) => {
  e.preventDefault();

  // Prevent duplicate submissions
  if (submitting) return;
  setSubmitting(true);

  try {
    const res = await fetch('/api/subscribe', { ... });

    if (res.ok) {
      // Fire Google Ads conversion
      gtag('event', 'conversion', {
        'send_to': 'AW-XXXXXXXXX/YYYYYYY',
        'value': 499.0, // Expected value of the lead
        'currency': 'USD'
      });
      setSuccess(true);
    }
  } catch (err) {
    setError(err.message);
  } finally {
    setSubmitting(false);
  }
};

Step 3: Fix the Duplicate Email Problem

Add idempotency to prevent duplicate form submissions from triggering multiple emails.

// Client-side: disable button + debounce

<button
  disabled={submitting}
  onClick={handleSubmit}
>
  {submitting ? 'Submitting...' : 'Get Started'}

// Server-side: check for existing email before sending

export async function POST(req) {
  const { email } = await req.json();

  // Check if already submitted in last 5 minutes
  const recent = await db.query(
    'SELECT id FROM leads WHERE email = ? AND created_at > NOW() - INTERVAL 5 MINUTE',
    [email]
  );

  if (recent.length > 0) {
    return Response.json({ ok: true, deduplicated: true });
  }

  // Insert lead + send email
  await db.insert('leads', { email, source: 'google_ads' });
  await sendWelcomeEmail(email);
}

Email Drip Sequence — 7 Emails Over 14 Days
Email 1: Welcome + Immediate Value
Instant Trigger: Form submit
Subject: "You're in — here's what Castford does in the first 5 minutes"
Short, warm welcome. Show one specific feature (e.g., automated variance detection). Include a 30-second GIF or screenshot of the dashboard. Single CTA: "Explore the platform" or "Book a 15-min demo."
CTA: Explore Platform
Email 2: Pain Point — Manual Close Process
Day 2 Nurture
Subject: "Still closing books in spreadsheets? Here's the cost"
Lead with the pain: "Finance teams spend 10+ days on month-end close." Show how Castford automates reconciliation. Include a stat or mini case study. CTA: Watch a 2-min walkthrough video.
CTA: Watch Walkthrough
Email 3: Social Proof — Who's Using It
Day 4 Trust
Subject: "How [Company Type] cut their close time by 60%"
Customer story or use case. Focus on a specific company type (Series B SaaS, PE-backed, multi-entity). Include specific metrics: "3-day close instead of 12." CTA: See more customer stories.
CTA: Read Case Study
Email 4: Feature Deep-Dive — AI Copilot
Day 7 Education
Subject: "Ask your data anything — meet the AI Copilot"
Show the AI Copilot feature. Example queries: "What's our burn rate at current run rate?" "Show me revenue by segment vs last year." Include screenshots. CTA: Try a live demo.
CTA: Book Live Demo
Email 5: Competitor Comparison
Day 9 Differentiation
Subject: "Castford vs Anaplan — what $499/yr gets you"
Direct comparison chart: Castford vs Anaplan/Planful/Adaptive. Hit on price ($499 vs $100K+), implementation time (days vs months), and ease of use. Don't trash competitors — just show the gap. CTA: Start free trial.
CTA: Start Free Trial
Email 6: Urgency — Limited Pricing
Day 11 Urgency
Subject: "Early adopter pricing closes [Date] — lock in $499/yr"
Create genuine urgency around early-adopter pricing. Mention the price will increase. Include a countdown or date. Remind them what they get. CTA: Lock in pricing now.
CTA: Lock In Pricing
Email 7: Final — Personal Note
Day 14 Personal
Subject: "Quick question, [First Name]"
Short, personal email from the founder/CEO. Plain text (no HTML template). Ask one question: "What's holding you back?" or "Is there a specific feature you need?" Reply-to goes to a real inbox. This converts the highest because it feels human.
CTA: Just Reply
